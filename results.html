Results

Performance, successes, limitations, and the main lessons learned from the final QArm tic-tac-toe system.

Performance Placement Accuracy Vision Results Limitations Future Improvements

Overall performance

The project met its primary goals by producing an autonomous tic-tac-toe robot that integrates robotic motion, camera-based detection, and game logic into one working system. During testing, the QArm repeatedly picked pieces from the staging area, placed them into the correct board cells, and followed the tic-tac-toe algorithm to complete games against a human player.

Placement accuracy

One of the biggest improvements came from lowering and controlling the release position of the end effector. Early trials dropped tokens from too high above the board, which caused wobble and unstable landings. Adjusting the approach height and placement angle made the token placement more stable and repeatable.

Final board setup that reflects the more stable piece placement behavior achieved later in testing.

Vision results

Strong color contrast

Gold robot tokens provided much stronger contrast than the earlier gray token concept, which improved board recognition reliability.

Contour success

Contour-based detection worked well for separated tokens and supported centroid-based board-state interpretation.

Occlusion issues

Closely aligned or stacked tokens could still be interpreted as one object, and center-cell occupancy could partially hide back-row pieces.

Observed limitations

  • Some moves still required intermediate waypoints to avoid collisions with staged or already placed tokens.
  • The arm speed was acceptable but could still be tuned further.
  • Depth detection was explored but not kept in the final perception chain because the depth contours were too unreliable.
  • User interface testing was mostly limited to the project team.

Future improvements

The report suggests faster motion tuning, better calibration for variable lighting, stronger object detection for occlusions, expanded user-interface development, and possibly different difficulty settings. Even with those open areas, the final demonstration succeeded as a complete and reliable mechatronics system that can autonomously play tic-tac-toe.
